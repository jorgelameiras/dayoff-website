# CasaFresh Website

Marketing landing page for CasaFresh — vacation home cleaning and AC maintenance services in Florida.

## What It Does

A clean, modern marketing website that introduces CasaFresh to potential clients (Airbnb/VRBO hosts and property managers).

## Features

- Hero section with value proposition
- Services overview (cleaning + AC maintenance)
- Pricing display
- Contact/booking call-to-action
- Mobile-responsive design
- Multiple design iterations (`index.html`, `index-v2.html`)

## Tech Stack

- Pure HTML, CSS, JavaScript
- No build step required
- Lightweight and fast

## Getting Started

Open `index.html` directly in a browser — no server needed.

Or serve it locally:

```bash
npx serve .
```
